## Developing the code yourself

1. Fork this repository into one of your own through GitHub.
2. Develop using Android's Eclipse plugin. 
	a. Install Eclipse, with the Android plugin
	b. Create a new project from source
	c. Make some changes
3. Submit a pull request through GitHub to allow your changes to be reviewed before integration.
